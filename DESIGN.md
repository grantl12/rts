# Project: THE DEEP STATE
## A Satirical RTS for the Post-Truth Era

### 🏗️ 1. Core Architecture & Tech Stack
The game is built on Godot 4.4 (Client) and Python/Supabase (Backend), prioritizing deep systems over "AAA" graphics.
*   **Visual Style:** "Subpar" but Stylized. Low-poly/Pixelated 3D with a Tron-Legacy neon grid overlay that changes color based on faction control.
*   **The "Shadow" Persistence:** All unit veterancy, player upgrades, and "Global Conflict" stats are stored in a Supabase cloud DB.
*   **The Shared Map Loop:** Every map is built as a "Theater." Players experience the same map from opposing sides in linear-but-smart campaign arcs to maximize asset ROI.

### 🎖️ 3. The CoH-Inspired Unit System
Units are handled as Squads with directional cover and persistent "Soul."
*   **Unit Tree & Veterancy:** Map Cap: Units can only reach a certain rank per mission to prevent early-game "God-units."
*   **The Hall of Heroes:** Units that hit the Map Cap can be "Retired" to your permanent roster or promoted to the Executive Board for global buffs.
*   **The "Meat Grinder" (Oligarchy):** A structural power that instantly spawns a mob of 10 Conscripts—quantity over quality.
*   **The "Stimulus Drop" (Regency):** A logistical power that drops cash and a squad of Gravy Seals—quality through capital.

### 📈 4. Persistent Meta-Progression (The War Room)
Players spend "Legacy Points" between missions on a Three-Pillar Meta-Tree:
*   **The Unit Tree:** Unlock persistent gear (e.g., Tactical Scooters for Gravy Seals, Thermal Scopes for Park Rangers).
*   **The Structural Grid:** Persistent building buffs (e.g., Reinforced Sandbags, The Meat Grinder 2.0).
*   **The Shadow Powers:** Upgrading the "Game Enders."

### 🏛️ 5. The "Superweapon" Decision Points
These are high-impact global abilities that turn the tide of the match:
*   **The SCOTUS Gavel (Regency):** Smashes an area and "De-Zones" it, preventing enemy building/rebuilding for a set duration.
*   **The Epstein File Leak (Sovereign/Oligarchy):** A global information blackout. Reveals the entire map and disables all enemy Hero units and income for 45 seconds.
*   **The Fact Check (Support):** A localized area-of-effect that debunks "Stealth" and weakens "Viral" bonuses.

### 🗺️ 6. The "Mirror-Map" Campaign Strategy
*   **The Demo/Standard Map:** The Quad of Compliance.
*   **The Incident:** The "Martyr" (The guy who was killed) serves as the narrative catalyst.
*   **The Perspective Flip:**
    *   **Campaign 1 (Regency):** Focus on "Security" and "Auditing" the Plaza.
    *   **Campaign 2 (Frontline):** Return to the same map to "Leak the Footage" and protect the "Martyr's Statue."

### 🖥️ 7. Development Milestones
1.  **Greybox Phase:** Build the Quad map in Godot. Implement basic A* pathfinding and the "Audit" capture mechanic.
2.  **The "Meat vs. Scalpel" Test:** Balance the Oligarchy Conscript squad vs. the Regency Park Ranger.
3.  **Supabase Integration:** Connect the "Post-Mission" screen to the cloud DB to save the first "Veteran" unit.
4.  **Satire Pass:** Record the "Redacted" Advisor briefings and implement the 16-bit glitch UI.

### 📝 8. Developer’s Notes & Satire Values
*   **Zero-Inference Personalization:** The game should feel like it knows the player's history but never explains why.
*   **Adaptive Tone:** The Advisor's tone shifts from "Bureaucratic Calm" to "Total Panic" as the player's global influence grows.
*   **The Goal:** Make a game that is fun to play, fun to meme, and impossible to forget.
